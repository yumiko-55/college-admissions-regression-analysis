# College Admission Rates & Cost of Attendance Analysis

## Overview
This project analyzes 26 years of U.S. Department of Education data to investigate the relationship between college admission rates and the cost of attendance.

## Research Question
Is there a relationship between a college's admission rate and its cost of attendance?

## Tools
- R
- tidyverse
- ggplot2
- Linear Regression

## Methods
- Data cleaning and preprocessing
- Exploratory data analysis
- Linear regression modeling
- Data visualization

## Dataset
- U.S. Department of Education college data
- 26 years of institutional information

## Key Findings
This project examined trends between admission rates and college costs using regression analysis and visualized the relationship between these variables.

## Skills Demonstrated
- Data cleaning
- Statistical modeling
- Regression analysis
- Data visualization
- Communicating analytical findings
